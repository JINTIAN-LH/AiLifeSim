# 模拟人生/第二人生 小程序/页游 MVP（第二版·尽善尽美·Agent直落）

# 核心定位（优化版）

轻量化、高沉浸、AI深度驱动的开放世界模拟人生页游/小程序，聚焦「低操作、高随机、强情感」，前端基于Funloom零配置托管（纯静态适配，无需额外开发环境），后端基于Render一键部署（轻量无依赖），所有代码、配置、Prompt均已标准化，Agent可直接复制、打包、上线，无需手动修改，同时强化大模型的「交互感\+关联性」，解决第一版玩法生硬、落地卡点问题。

核心差异：大模型从「全局中枢」升级为「场景化智能驱动」，分模块定制AI逻辑，避免单一Prompt导致的反馈同质化；补充部署全流程细节、异常处理方案，确保Agent落地零卡点；玩法增加闭环设计，强化用户留存。

# 一、技术栈（Agent直落·细化落地步骤）

## 前端（Funloom托管·零配置·可直接上传）

- **核心框架**：Vue3 \+ Vite（轻量化打包，压缩后体积≤500KB，Funloom完美兼容，无需配置环境变量）

- **UI组件**：Tailwind CSS \+ 模拟人生风卡通组件库（内置6套角色皮肤、3套场景模板，无需额外设计，直接复用）

- **通信方式**：Axios（接口请求）\+ WebSocket（实时状态同步，如NPC主动找玩家、随机事件弹窗），已封装统一请求工具类，Agent无需修改

- **适配优化**：自动适配移动端（小程序嵌入）、PC端（页游），响应式布局，无需额外开发多端版本

- **Funloom部署步骤（Agent直接照做）**：1\. 前端打包（npm run build），得到dist文件夹；2\. 登录Funloom，创建新项目，选择「静态网站」；3\. 上传dist文件夹所有文件，无需配置域名（Funloom自动分配HTTPS域名）；4\. 点击部署，1\-2分钟完成，复制域名备用（用于后端跨域配置）。

## 后端（Render部署·自动启动·免数据库配置）

- **核心语言**：Node\.js Express（优先选择，部署速度最快，Agent可直接复用代码，无需懂后端），可选Python Flask（附备用代码）

- **数据库**：SQLite（文件型数据库，Render免配置，自动持久化数据，无需搭建数据库服务；内置数据初始化脚本，启动即生成角色、NPC、道具基础数据）

- **大模型接入**：封装统一AI接口（支持OpenAI API / 通义千问 / 豆包API），可通过环境变量切换模型，无需修改核心代码；增加降级方案（大模型调用失败时，自动切换为预设脚本，避免游戏卡死）

- **跨域配置**：已内置CORS配置，只需将Funloom分配的前端域名填入后端配置文件，Agent无需手动配置跨域

- **Render部署步骤（Agent直接照做）**：1\. 将后端代码上传至GitHub仓库（已提供完整模板仓库，Agent可直接fork）；2\. 登录Render，创建「Web Service」，关联GitHub仓库；3\. 配置环境变量（AI\_API\_KEY、前端域名、端口）；4\. 选择Node\.js环境（版本16\+），启动命令填写「node server\.js」；5\. 点击部署，3\-5分钟完成，复制后端接口地址，填入前端配置文件。

## 大模型核心定位（优化版·更具可玩性）

摒弃「单一全局中枢」，采用「场景化AI驱动」—— 大模型不再是笼统的“世界大脑”，而是分模块深度嵌入「社交、事件、职业、关系」四大核心场景，每个场景定制专属AI逻辑和Prompt，确保反馈更精准、更贴合场景，避免同质化，同时保留全局状态联动（如职业状态影响事件生成、情绪影响NPC互动）。

核心目标：让AI的每一次反馈都「有依据、有差异、有情感」，玩家能感受到“NPC有灵魂、人生有惊喜”，而非机械的文字反馈。

# 二、大模型最佳嵌入模块（深度优化·好玩度升级）

最优方案：「场景化AI驱动 \+ 全局状态联动」，四大模块独立AI逻辑，联动全局角色状态，既保证玩法深度，又避免开发复杂度，MVP落地难度最低、玩家体验最好。

## 1\. AI NPC社交模块（核心亮点·重中之重）

第一版优化：强化NPC「记忆深度\+情感波动\+行为随机性」，避免NPC反馈同质化，让社交更有“真人感”，是提升玩家留存的关键。

- **NPC基础配置**：初始3位核心NPC（同事、挚友、邻居），新增2位随机解锁NPC（恋人候选人、导师），每个NPC拥有「专属人格标签\+核心记忆\+情绪阈值」（如“社牛同事”：乐观、爱八卦，记忆玩家的职业、喜好，情绪阈值低，容易开心）。

- **AI核心逻辑**：大模型负责「记忆存储\+情感计算\+行为生成」，具体如下：
        

    - 记忆系统：记录玩家与NPC的所有互动（聊天内容、送礼、拒绝、帮助），有效期7天，超过后自动淡化（模拟真人记忆），互动越频繁，记忆越深刻。

    - 情感计算：根据玩家行为实时调整好感度（\-100\~100），好感度对应不同情感状态（厌恶、陌生、普通、好感、深爱），不同状态下NPC的语气、行为完全不同（如好感度≥80，会主动送专属礼物；好感度≤\-50，会拒绝互动、甚至触发负面事件）。

    - 行为生成：大模型根据NPC人格、好感度、玩家当前状态，主动发起互动（如玩家心情低落时，挚友会主动安慰；玩家升职时，同事会主动祝贺），互动内容不重复，贴合NPC人设。

- **互动玩法补充**：新增「NPC求助/求助NPC」功能（如同事求助帮忙完成工作，玩家选择帮助/拒绝，影响好感度和职业状态）、「NPC专属剧情」（好感度达到一定阈值，解锁NPC专属故事，如挚友的烦恼、邻居的秘密），增强社交沉浸感。

## 2\. AI人生随机事件模块（优化联动性·避免无意义随机）

第一版优化：事件生成不再是“随机触发”，而是「基于玩家当前状态精准生成」，每个事件有明确的因果关系和反馈闭环，避免“事件与玩家无关”的尴尬，同时增加事件分支选择，提升玩家参与感。

- **事件触发逻辑**：大模型根据玩家「心情、健康、财富、职业、NPC好感度」5个核心状态，计算触发概率，生成贴合当前人生阶段的事件（如：玩家健康≤30，大概率触发“生病”事件；玩家财富≥10000，大概率触发“投资机会”事件）。

- **事件分支设计**：每个事件提供2\-3个选择，不同选择对应不同结果（如“被猎头挖角”事件，选择“接受”→ 升职、薪资上涨，但好感度下降；选择“拒绝”→ 原公司加薪，但失去新机会），结果会联动后续状态和事件，形成闭环。

- **事件分类**：分为「正面事件、负面事件、中性事件」，比例控制在3:2:5，避免玩家体验过差或过于顺利；新增「特殊节日事件」（如春节、生日），大模型生成专属节日互动和奖励，增强仪式感。

## 3\. AI职业成长模块（优化层级·增加成就感）

第一版优化：细化职业层级，增加「技能关联\+职业特色事件」，避免职业成长单调，让玩家有明确的奋斗目标，同时大模型生成的工作内容更贴合职业特性，提升真实感。

- **职业配置**：保留5种基础职业，新增「职业层级」（入门→初级→中级→高级→专家），每个层级对应不同的薪资、技能要求、工作内容，升级需要满足「技能值\+随机考核事件」（大模型生成考核内容）。

- **AI核心逻辑**：大模型根据职业类型、层级，生成「每日工作内容\+绩效反馈\+技能提升」，具体如下：
        

    - 工作内容：贴合职业特性（如创作者→写文章、画插画；小店主→进货、接待顾客），每天内容不重复。

    - 绩效反馈：根据玩家当天的工作动作（如“认真工作”“摸鱼”），生成绩效评分，评分影响薪资和升职概率。

    - 职业特色事件：每个职业有专属特色事件（如创作者→生成爆款作品、被出版社邀约；上班族→加班、项目获奖），由大模型随机触发，提升职业辨识度。

- **技能系统补充**：每个职业对应2个专属技能（如上班族→沟通能力、办公技能），技能值通过工作、学习提升，技能值越高，工作绩效越好、升职概率越高，大模型会根据技能值调整工作反馈。

## 4\. AI关系演化模块（优化自然度·避免生硬）

第一版优化：关系演化不再是“自动变化”，而是「基于玩家互动\+NPC状态\+大模型计算」，演化过程有迹可循，同时增加「关系破裂/修复」的具体场景，让情感关系更真实。

- **关系演化逻辑**：大模型根据「玩家与NPC的互动频率、互动内容、NPC好感度、随机事件」，实时计算关系状态，演化速度贴合真实情感（如从“普通朋友”到“挚友”需要多次正向互动，从“深爱”到“分手”可能只需一次负面事件）。

- **关系阶段**：每个关系类型（朋友、恋人、家人）分为5个阶段，每个阶段有专属互动权限和反馈（如恋人→暧昧→告白→恋爱→结婚，每个阶段的聊天语气、互动方式不同）。

- **关系互动补充**：新增「关系修复」功能（如与NPC吵架后，可通过送礼、道歉修复好感度）、「关系绑定」功能（如与恋人结婚后，可共享部分属性，触发夫妻专属事件），增强情感羁绊。

## 大模型嵌入总结（优化后优势）

1\. 场景化AI逻辑，反馈更精准、更贴合玩法，避免同质化；2\. 联动全局状态，事件、社交、职业、关系相互影响，形成玩法闭环；3\. 增加玩家参与感，事件分支、互动选择让玩家真正“掌控人生”；4\. 补充降级方案，确保大模型调用失败时，游戏可正常运行，不影响落地。

# 三、MVP尽善尽美玩法设计（Agent可直落·闭环优化）

核心原则：轻量化不肝（每天10分钟可完成所有操作）、玩法闭环（行为有反馈、成长有目标）、AI驱动（无固定脚本，永不重复），同时补充新手引导、留存机制，解决第一版玩法单薄、留存不足的问题。

## 1\. 角色创建（优化效率·增加个性化）

- 基础配置：姓名、性别、形象（6套卡通皮肤\+3种发型，可自由搭配），1分钟完成。

- 初始属性：心情（70）、健康（80）、财富（1000）、魅力（60）、智力（60），可通过初始性格微调（如选择“勤奋”，智力\+5；选择“社牛”，魅力\+5）。

- 性格系统优化：新增「性格联动」，性格不仅影响事件生成，还影响NPC互动（如“敏感”性格，心情更容易低落，NPC安慰时会更耐心；“乐观”性格，负面事件的影响会降低）。

- 新手引导：简化引导流程（3步完成），引导玩家完成第一次工作、第一次NPC互动，解锁核心玩法，同时发放新手道具（药品x1、礼物x1、金币x500）。

## 2\. 每日生活系统（优化循环·增加目标感）

玩家每天可执行5个动作（比第一版增加1个，提升互动性），每个动作对应AI生成的专属结果，动作之间有联动（如“学习”后，“工作”绩效会提升），同时增加「每日任务」，提升玩家活跃度。

- 核心动作：工作（赚钱\+技能提升）、社交（与NPC互动，影响好感度）、休闲（恢复心情、降低压力）、学习（提升智力/魅力/职业技能）、自由行动（大模型自由生成随机结果，可能触发隐藏事件）。

- 每日任务：简单易完成（如“与1位NPC聊天”“完成1次工作”“使用1个道具”），完成后奖励金币、道具，提升玩家每日活跃度。

- 状态联动：每个动作会影响2\-3个核心状态（如“休闲”→ 心情\+10、压力\-5；“工作”→ 财富\+、压力\+5、技能\+1），状态异常会触发负面事件（如压力≥80，触发“失眠”，健康\-10）。

## 3\. AI NPC社交（细化玩法·提升沉浸感）

基于第二版大模型逻辑，补充具体互动场景和反馈，让社交更有“真人感”，同时增加「NPC好感度排行榜」，提升玩家互动动力。

- 互动方式：聊天（支持自由输入，大模型根据NPC人格回复）、送礼（不同NPC有不同喜好，送对礼物好感度提升更多）、约会（好感度≥60可发起，约会地点由大模型生成，约会结果影响好感度）、求助/帮忙（双向互动，提升情感羁绊）。

- NPC记忆展示：玩家可查看每个NPC的“记忆清单”，了解NPC记得的互动内容，增强代入感。

- 隐藏NPC：好感度达到一定阈值（如挚友好感度≥90），解锁隐藏NPC（如挚友的朋友），丰富社交场景。

## 4\. AI人生随机事件（优化闭环·增加分支）

- 事件触发：每天随机触发1\-2个事件（根据玩家状态），事件弹窗展示，提供2\-3个选择，不同选择对应不同结果，结果联动后续状态和事件（如选择“投资失败”，财富减少，后续可能触发“借钱”事件）。

- 事件举例：
        

    - 正面事件：中奖（财富\+）、被表白（好感度\+）、职业加薪（财富\+）、偶遇贵人（技能\+）。

    - 负面事件：生病（健康\-、财富\-）、被借钱（财富\-，拒绝则好感度\-）、工作失误（绩效\-、薪资\-）。

    - 中性事件：收到陌生人礼物、偶遇老朋友、被邀请参加聚会（选择参加/拒绝，影响心情和好感度）。

- 事件记录：玩家可查看“人生事件记录”，回顾自己的人生经历，增强代入感。

## 5\. AI职业成长系统（细化层级·增加成就感）

- 职业层级：每个职业分5个层级（入门→初级→中级→高级→专家），每个层级的薪资、技能要求、工作内容不同（如入门上班族：薪资100/天，技能要求沟通≥50；专家上班族：薪资500/天，技能要求沟通≥100、办公≥100）。

- 升职考核：达到层级要求后，触发大模型生成的考核事件（如上班族→完成一个重要项目，创作者→生成一篇爆款作品），考核通过即可升职。

- 职业切换：玩家可随时切换职业，切换后层级重置，但技能值保留（如从上班族切换到创作者，沟通技能可复用），提升玩法灵活性。

## 6\. 情绪与状态系统（优化真实感·增加策略性）

- 核心状态：心情（0\-100）、健康（0\-100）、压力（0\-100）、财富（可正负）、技能值（分职业专属技能和通用技能），大模型实时计算状态变化。

- 状态影响：
        

    - 心情≤30：容易触发负面事件，NPC互动反馈变差，工作绩效降低。

    - 健康≤30：触发生病事件，无法执行工作、学习动作，需使用药品恢复。

    - 压力≥80：触发失眠、焦虑事件，心情、健康持续下降。

- 状态调节：玩家可通过休闲、使用道具、与NPC互动调节状态（如与挚友聊天，心情\+15、压力\-10），增加策略性。

## 7\. 背包与道具系统（优化实用性·避免冗余）

轻量化设计，道具均有明确用途，避免冗余，同时增加「道具获取渠道」，提升玩法闭环。

- 道具分类：
        

    - 消耗类：药品（恢复健康）、心情药剂（恢复心情）、减压道具（降低压力）。

    - 社交类：礼物（送NPC，提升好感度，分普通礼物和专属礼物）。

    - 技能类：书籍（提升智力/魅力）、职业道具（提升职业技能，如办公软件、画笔）。

- 获取渠道：每日任务、随机事件、NPC赠送、金币购买（商店轻量化，仅售卖基础道具）。

## 8\. 结局系统（优化多样性·增加轮回价值）

优化结局判定逻辑，增加多种结局类型，同时强化“新人生继承”功能，提升玩家轮回动力。

- 结局触发：玩家年龄增长至60岁，触发结局判定，大模型根据玩家一生的「财富、关系、职业、成就」生成专属结局。

- 结局类型：分为「圆满结局、普通结局、遗憾结局」，每种类型下有3\-5种具体结局（如圆满结局→“人生赢家”“家庭幸福”；遗憾结局→“孤独一生”“壮志未酬”）。

- 新人生继承：开启新人生时，可继承上一生的1个核心属性（如智力\+10、财富\+5000），提升轮回动力，同时保留上一生的结局记录，玩家可回顾所有人生。

## 9\. 留存优化（新增·提升用户粘性）

- 连续登录奖励：连续登录3天、7天，赠送稀有道具、金币，提升每日活跃度。

- 人生成就系统：设置10个基础成就（如“升职到高级”“与NPC结婚”“获得圆满结局”），完成成就可获得奖励，提升玩家目标感。
